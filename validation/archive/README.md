# validation/archive/

Historical VM-smoke output, preserved for reference but moved out of
`validation/` root to keep the live harness visible.

Each subdirectory is dated (YYYY-MM-DD). Files within are a snapshot of the
`fresh-vm-smoke.sh` output for that run. To regenerate: use the script in
`validation/fresh-vm-smoke.sh` with the same per-distro fixtures.

These files are not regression inputs — they are evidence. If you need to
compare a new run against these, diff by hand.

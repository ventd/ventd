# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

Be respectful. Focus on the technical question. Hardware compatibility threads can get frustrating — assume good faith, debug collaboratively, and credit reporters when their logs help fix a bug.

Harassment, personal attacks, and discriminatory language are not tolerated. Violations will result in warnings or bans at the maintainers' discretion.

## Reporting

Report violations to the maintainers via the email listed on the organisation's GitHub page, or via GitHub's own abuse reporting tools. Reports will be handled privately.

## Enforcement

The Contributor Covenant enforcement ladder applies: correction → warning → temporary ban → permanent ban, depending on severity and pattern.
